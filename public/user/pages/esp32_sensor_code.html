传感器固件代码
复制以下代码到 Arduino IDE，烧录到 ESP32 即可作为传感器使用
使用说明：
1. 安装 Arduino IDE，添加 ESP32 开发板支持
2. 安装库：PubSubClient（MQTT）、ArduinoJson（v6）
3. 按下方接线连接土壤湿度传感器：AO 接 GPIO34，供电脚接 GPIO25
4. 烧录后首次启动会打开 AP 热点 木白云IoT-传感器（密码 12345678），连接后自动弹出配网页面
5. 配网成功后自动通过MQTT注册设备码并上报土壤湿度数据，设备码以 C 开头
默认接线

     可修改的配置项：


    MQTT_SERVER — MQTT服务器地址，如 mqtt.mcoud.cn

    MQTT_PORT — MQTT端口，默认 1883


    土壤湿度接线：

    SOIL_SENSOR_PIN — 土壤湿度 AO 引脚，默认 GPIO 34

    SENSOR_POWER_PIN — 传感器供电控制脚，默认 GPIO 25


    校准参数：

    DEFAULT_DRY_ADC — 干燥 ADC，默认 2320

    DEFAULT_WET_ADC — 湿润 ADC，默认 475

    NUM_READINGS — 每次采样次数，默认 30


    休眠周期：

    DEFAULT_SLEEP_SECONDS — 默认休眠时间，默认 30秒；平台 /user/pages/sensor_setting 保存后会通过 MQTT 下发新的休眠时间


     恢复出厂：

    串口发送 FACTORY_RESET 即可清空配网信息，重新进入AP配网模式

    串口发送 SHOW_CAL 可查看当前 Dry/Wet 校准值
